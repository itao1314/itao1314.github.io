Swift-基本运算符

发表于 2019-05-27 | 分类于 iOS开发

赋值运算符

  1. 赋值的右边是一个多元组，它的元素会被分解多个常量和变量。

  2. Swift的赋值操作并不返回任何值。C和OC是返回值得。

算术运算符

  1. Swift默认情况下不允许在数值运算中出现溢出的情况。

组合赋值运算符

  1. let b = a += 2是错误的，因为复合赋值运算没有返回值，a += 2是没有返回值的。

比较运算符

  1. Swift 也提供恒等（===）和不恒等（!==）这两个比较符来判断两个对象是否引用同一个对象实例。

  2. 元组的元素相同，且长度相同，元组就可以被比较。元组的比较会从左到右逐个比较，知道发现有两个值相等时停止。

  3. Bool不能被比较，所以含有布尔类型的元组不能被比较。

  4. Swift 标准库只能比较七个以内元素的元组比较函数。如果你的元组元素超过七个时，你需要自己实现比较运算符。

空合运算符

  1. 空合运算符(a ?? b)将对可选类型a进行空判断，如果a包含一个值就解包，否则返回一个默认值b。表达式 a 必须是 Optional 类型。默认值 b 的类型必须要和 a 存储值的类型保持一致。

区间运算符

  1. 闭区间运算符(a...b)定义一个从a到b的所有值的区间。

  2. 半开区间运算符(a..<b)定义一个从a到b但不包含b的区间。

单侧区间

  1. 闭区间操作符有另一表达形式，表达往一侧无限延伸的区间。

  2. 半开区间操作符也有单侧表达式。

  3. 不能遍历省略了初始值的单侧区间，因为遍历的开端并不明显。
