Swift-集合类型

发表于 2019-05-28 | 分类于 iOS开发

  1. Swift用Array、Sets、Dictionaries三种基本的集合类型用来存储集合数据。

  2. Array是有序数据的集。

  3. Sets是无序无重复数据的集。

  4. Dictionaries是无序的键值对的集。
